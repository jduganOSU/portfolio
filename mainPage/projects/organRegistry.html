Project Description

The Organ Donation Registry System is a comprehensive web application designed to manage and streamline the process of organ donation and transplantation. The application provides a user-friendly interface for managing donors, recipients, organs, and transplants, and includes a matching feature to pair organ donors with recipients.

Key Features

  • Donor and Recipient Management
  • Organ Management
  • Transplant Management
  • Organ-Recipient Matching
  • User-Friendly Interface

Technical Stack:

  • Built using the Express.js framework for Node.js.
  • Employed Handlebars.js as the templating engine for dynamic HTML generation.
  • Utilized AJAX for asynchronous data fetching and manipulation, allowing for seamless updates without page reloads.
  • Follows MVC architecture for a clear separation of concerns (models, views, controllers).
  • Styles managed with CSS for a consistent and appealing visual design.

Role and Responsibilities:

  • Acted as a full-stack developer, overseeing both front-end and back-end development.
  • Implemented server-side logic using Express.js.
  • Designed the database schema and authored queries for data manipulation.
  • Developed the user interface using Handlebars.js and CSS.
  • Managed AJAX requests for dynamic data updates.
  • Ensured the application adhered to best practices in code organization, error handling, and security.

Project Outcomes

The Organ Donation Registry System has greatly improved the efficiency and accuracy of managing organ donations and transplants. The system's user-friendly interface and automated matching feature have reduced manual work and potential errors, leading to a smoother and faster transplantation process.
